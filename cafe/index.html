Website Deployment Guide

  1. Create a GitHub repository named "cafe-1958"
  2. Upload all files to the repository
  3. Create an "images" folder and add your background images
  4. Enable GitHub Pages in repository settings
  5. Visit: https://[your-username].github.io/cafe-1958/

For local use: Download all files and open index.html in a browser
